# คู่มือภาษากลางของ NurseGo

เอกสารนี้ใช้กำหนดโทนภาษาในแอปให้สม่ำเสมอ ทั้งหน้าผู้ใช้ทั่วไป หน้าพยาบาล หน้าสถานพยาบาล และหน้าผู้ดูแลระบบ

## หลักการใช้ภาษา

- ใช้ภาษาไทยที่อ่านง่าย ชัด และเป็นมิตร
- เน้นความปลอดภัย ความเป็นส่วนตัว ความสะดวก และความรวดเร็ว
- อธิบายสิ่งที่ผู้ใช้จะได้รับ มากกว่าบอกว่าอะไร "ถูกบล็อก"
- ถ้าเป็นข้อความหลังบ้าน ให้ใช้คำไทยที่ชัดเจนแทนคำเทคนิคอังกฤษเมื่อทำได้
- ถ้าเป็นข้อความกฎหมาย ให้ใช้ภาษาทางการ ชัดเจน และไม่ลดความหมายทางกฎหมาย

- ประกาศงาน
- ผู้ช่วยขึ้นเวรแทน
- เริ่มคุยผ่านแอป
- ช่องทางพูดคุย
- แสดงความสนใจ
- สิทธิ์การใช้งาน
- แพ็กเกจ
- ช่วงใช้ฟรี
- เปิดระบบชำระเงิน
- ประเภทบัญชี
- ผู้ดูแลระบบ
- ข้อความสำเร็จรูป
- กลุ่มเป้าหมาย
- ตั้งเวลาส่ง

## งานโพสต์

ควรใช้

- ประกาศงานเวร
- ประกาศหาผู้ช่วยขึ้นเวรแทน
- ระบุวัน เวลา และรายละเอียดงาน
- เปิดประกาศได้ทันที

ควรเลี่ยง

- ขายเวร
- แลกเวร
- รับคนแทนแบบซื้อขาย

ตัวอย่าง

- ใช้: "ประกาศหาผู้ช่วยขึ้นเวรแทนเฉพาะวัน"
- เลี่ยง: "ขายเวรด่วนวันนี้"

## การติดต่อ

ควรใช้

- เริ่มคุยผ่านแอป
- คุยผ่านแอปก่อนเพื่อความเป็นส่วนตัว
- เลือกช่องทางพูดคุยที่สะดวก
- ข้อมูลติดต่อจะแสดงในขั้นตอนที่เหมาะสม

ควรเลี่ยง

- ซ่อนเบอร์ไว้ก่อน
- ต้องกดรับงานก่อน
- ปลดล็อกข้อมูลติดต่อ
- ห้ามส่งเบอร์

ตัวอย่าง

- ใช้: "เริ่มคุยผ่านแอปก่อนเพื่อความสะดวกและเป็นส่วนตัว"
- เลี่ยง: "ประกาศนี้ให้คุยในแอปก่อน"

## สิทธิ์บัญชีและแพ็กเกจ

ควรใช้

- สิทธิ์การใช้งาน
- เปิดใช้รายครั้ง
- ช่วงทดลองใช้ฟรี
- เปิดระบบชำระเงิน
- แพ็กเกจที่เหมาะกับการใช้งาน

ควรเลี่ยง

- ร้านค้า
- ซื้อบริการ
- monetization
- early access
- live billing

ตัวอย่าง

- ใช้: "เปิดใช้รายครั้งสำหรับประกาศนี้"
- เลี่ยง: "ซื้อแยกรายครั้ง"

## ข้อความกฎหมาย

หลักการ

- ใช้ภาษาทางการ
- คงความหมายทางกฎหมายให้ครบ
- ไม่แต่งให้ดูอ่อนลงจนสาระสำคัญเปลี่ยน

ควรใช้

- เงื่อนไขการใช้งาน
- นโยบายความเป็นส่วนตัว
- การชำระเงินเป็นที่สิ้นสุด
- ตามที่กฎหมายกำหนด
- ตามสภาพการให้บริการ

ควรเลี่ยง

- คำโฆษณาเกินจริง
- คำรับประกันที่ไม่มีฐานทางกฎหมาย
- ภาษากึ่งเล่นกึ่งทางการในข้อกำหนดสำคัญ

## รูปแบบประโยคที่แนะนำ

- ใช้ประโยคสั้น กระชับ
- ขึ้นต้นด้วยประโยชน์หรือสิ่งที่ผู้ใช้ทำได้
- ถ้าต้องแจ้งข้อจำกัด ให้บอกเหตุผลเชิงดูแลผู้ใช้

ตัวอย่าง

- ใช้: "คุยผ่านแอปก่อนเพื่อความเป็นส่วนตัว"
- ใช้: "ข้อมูลนี้จะแสดงในขั้นตอนที่เหมาะสม"
- เลี่ยง: "ระบบไม่อนุญาตให้..."

## เช็กลิสต์ก่อนเพิ่มข้อความใหม่

- ข้อความนี้อ่านแล้วเป็นภาษาคนทั่วไปหรือยัง
- มีคำเทคนิคอังกฤษที่แปลเป็นไทยได้หรือไม่
- ถ้อยคำนี้สื่อเรื่องความปลอดภัยและความเป็นส่วนตัวอย่างเหมาะสมหรือไม่
- มีคำที่เสี่ยงเชิงกฎหมายหรือทำให้ตีความว่าเป็นการซื้อขายเวรหรือไม่
- ถ้าเป็นข้อความกฎหมาย ได้คงความหมายทางกฎหมายครบหรือไม่
